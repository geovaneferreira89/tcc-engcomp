\section{Objetivos}

O objetivo geral deste projeto é desenvolver um sistema para reconhecimento do complexo QRS em sinais biomédicos de eletrocardiografia (ECG). Para melhor definição do escopo, separamos nos seguintes objetivos específicos:  

%+ tabulação e - espaçamento
\begin{itemize}
	\item Desenvolvimento de um ambiente para visualizar sinais de  ECG armazenados em arquivos digitais;
	\item Implementar no ambiente a funcionalidade de marcar eventos no sinal e salvá-los para uso futuro;
	\item Reconhecimento de padrões, realizado a partir da extração de características das marcações salvas.
\end{itemize}
